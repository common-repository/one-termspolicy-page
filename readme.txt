== Description ==
**افزونه ایجاد شرایط و قوانین وینداد**
بسته کامل جهت ایجاد رایگان انواع شرایط ، قوانین و حریم خصوصی به صورت کاملا اتوماتیک
از جمله :
قوانین وبسایت فروشگاه کالا ، قوانین وبسایت فروشگاه خدماتی ، قوانین وبسایت شخصی

= Features =
* بسته کامل ایجاد قوانین و شرایط
* قوانین کاملا بروز
* ایجاد قوانین به صورت اتوماتیک
* ایجاد شورت کد برای هر قانون و استفاده دلخواه از آن در هرکجا
* امکان اجبار کاربر هنگام ورود جهت قبول قانون انتخابی توسط شما
* امکان اجبار کاربر هنگام ثبت نام جهت قبول قانون انتخابی توسط شما
* ایجاد قوانین وبسایت فروشگاه کالا
* ایجاد قوانین وبسایت فروشگاه خدماتی
* ایجاد قوانین وبسایت شخصی
* انتخاب قوانین ایجاد شده در بخش حریم خصوصی
* و امکانات فراوان دیگر


= Compatibility =
* آخرین نسخه وردپرس
* جهت دریافت مشاوره حقوقی کسب‌وکار [vindad.com](https://vindad.com)
